# ecommerce-backend-system
Production-ready E-Commerce backend application built using Java, Spring Boot, Spring Security, JPA, MySQL, Redis, and Docker following clean architecture and RESTful API principles.



## Features

- JWT Authentication & Authorization
- Role-Based Access Control
- User Management
- Product Management
- Category Management
- Shopping Cart Functionality
- Order Processing
- Payment Integration

- RESTful APIs
- Global Exception Handling
- Input Validation
- Pagination & Sorting


---

### Backend
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate

- MySQL
